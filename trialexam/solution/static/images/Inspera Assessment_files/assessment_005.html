Hvilke språk skal støttes i denne testen?

Hver oppgave kan redigeres i språkene som støttes. Nye oppgaver blir opprettet på det aktive språket, forfatteren må deretter oversette teksten til alle aktuelle språk.

Det er bare teksten i oppgavene som må oversettes. Alle endringer på innstillinger (slik som struktur, poeng og namn) er generelle for oppgaven og blir ikke påvirket av språkvalget.
